# \<windhappers-client-web\>

Web application for canoe association De Windhappers

## Build

```sh
./build.sh
```

## Run

### Development

```sh
npm start
```

### Docker

```sh
docker run --rm --interactive --tty --publish 8888:80 xsystems/windhappers-client-web
```
